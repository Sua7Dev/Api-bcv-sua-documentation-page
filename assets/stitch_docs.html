Getting Started

Welcome to the API SUA-BCV documentation. Our API provides real-time access to regional exchange rates and currency conversion data in a highly performant JSON format.

API Configuration

GET https://api-sua-bcv/v1/
Format: JSON

key Authentication

To access the API, you must include your API key in the x-api-key header of every request.

warning

Warning

Never share your API keys publicly. Use environment variables to store them securely in your production environments.

info

Note

Rate limits apply: 1,000 requests per minute for free tier users.

Request Examples

const response = await fetch('https://api-sua-bcv/v1/rates', {
  headers: {
    'x-api-key': 'YOUR_API_KEY',
    'Accept': 'application/json'
  }
});

const data = await response.json();
console.log(data);
