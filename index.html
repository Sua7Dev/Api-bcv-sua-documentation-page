API SUA-BCV: Datos cambiarios en tiempo real

Optimizado para Bun y desplegado en el Edge. Obtén las tasas oficiales del BCV con latencia mínima y máxima confiabilidad.

example.js
const response = await fetch('https://api-sua-bcv.io/v1/usd', {
headers: { 'Authorization': 'Bearer YOUR_API_KEY' }
});
const data = await response.json();
console.log(data);
// Output:
{
  "rate": 36.42,
  "currency": "USD",
  "updated_at": "2024-05-20T14:00:00Z"
}
