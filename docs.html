Getting Started

Bienvenido a la documentación oficial de SUA-BCV. Nuestra API proporciona acceso en tiempo real a las tasas de cambio regionales y datos de conversión de moneda en un formato JSON altamente eficiente y optimizado.

Configuración de la API
GET https://api-sua-bcv.io/v1/
Formato: JSON

key Autenticación

Para acceder a la API, debes incluir tu clave de API en el encabezado x-api-key de cada solicitud.

warning
Atención

Nunca compartas tus claves de API públicamente. Utiliza variables de entorno para almacenarlas de forma segura en tus entornos de producción.

info
Nota

Se aplican límites de tasa: 1,000 solicitudes por minuto para usuarios del nivel gratuito.
